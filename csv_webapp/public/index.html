📝 How to use this platform

  • Select a Movie: Click any "Not Started" or "In Progress" movie below to open the editor.
  • Edit Actors: Once loaded, click any actor's name to modify or correct the tags.
  • Auto-save: All your changes are automatically saved to the database in real-time as you type.
  • Push to Cloud: When you are completely done editing, click the green Push to Cloud button to permanently save the CSV across GitHub.
  • Status: Movies marked as Complete are locked securely and cannot be edited further.
